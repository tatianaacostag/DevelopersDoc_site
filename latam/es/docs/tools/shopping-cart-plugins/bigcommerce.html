BigCommerce

Esta guía ofrece instrucciones detalladas para integrar PayU con tu tienda de BigCommerce, permitiendo soluciones de pago eficientes.

BigCommerce es una plataforma de comercio electrónico robusta diseñada para ayudarte a configurar y administrar una tienda en línea rápidamente, ofreciendo una variedad de funciones listas para usar para una experiencia sin interrupciones. Para más información, visita el sitio web oficial de BigCommerce.

Requisitos Previos

Antes de comenzar la integración, asegúrate de tener lo siguiente:

  • Una cuenta de PayU Latam
  • Una cuenta de BigCommerce

Instalación de la Extensión

  1. Ve al Marketplace de BigCommerce, selecciona My Apps, localiza la extensión de PayU y haz clic en Install.

  1. Marca la casilla de verificación y haz clic en Confirm para continuar con la instalación.

  1. Después de la instalación, la interfaz de la aplicación se verá de la siguiente manera.

Configuración de Métodos de Pago

Configura y personaliza los métodos de pago para tu página de pago. Consulta nuestros métodos de pago disponibles.

  1. Después de la instalación, activa el interruptor Enable PayU Latam para integrar la aplicación con tu tienda de BigCommerce.

  1. Personaliza el título para la opción de pago con tarjeta (por ejemplo, Pagar con tarjeta). Este título será visible para los usuarios en la página de pago.

  1. Elige el formato de visualización para las opciones de pago con tarjeta disponibles en el checkout:

    • Mostrar Powered by PayU Latam y el logo en el formulario de pago.
    • Mostrar los logos de tarjetas de crédito en la página de pago.

Importante

Para habilitar métodos de pago alternativos (efectivo y transferencias bancarias), selecciona Enable Web Checkout (APM).
  1. Agrega un proveedor para un país específico de PayU Latam haciendo clic en Add Provider. Activa o desactiva cada proveedor con el botón en la columna Status y actualiza sus credenciales con el botón Edit.

Además, elige el modo Test o Live para cada proveedor en Onboard Setup.

Campos Obligatorios para Agregar un Nuevo Proveedor:

Parámetro Descripción
Select Payu Latam country Elige el país que deseas configurar.
API Login Usuario o login que PayU te proporciona. Cómo obtener mi API Login.
API Key Clave única que PayU asigna a tu tienda. Cómo obtener mi API Key.
Public Key Clave única que PayU asigna a tu tienda. Cómo obtener mi Public Key.
Account ID ID de la cuenta de PayU para el país seleccionado.
Merchant ID ID de tu comercio en PayU Latam.

  1. Selecciona el flujo de transacción:
  • Para habilitar el flujo de un paso, selecciona Capture on Order Placed.
  • Para habilitar el flujo en dos pasos, selecciona Capture on Shipment.

Importante

  • El flujo de dos pasos se aplica solo a los pagos con tarjeta (sujeto a disponibilidad en cada país). Otros métodos de pago requieren autorización y captura en un solo paso. Consulta la documentación de flujos de pago para más detalles.
  • Las opciones de múltiples tiendas aparecerán debajo de la sección de opciones de pago.

Gestión de Pedidos y Reembolsos

  1. Para gestionar los pedidos, haz clic en el botón Order Dashboard en la esquina superior derecha.

  1. El Order Dashboard proporciona una vista general del estado de los pagos para todos los pedidos.

  1. Para procesar un reembolso, haz clic en el botón Refund en la columna de Actions.

Importante

  • Los reembolsos están disponibles solo para transacciones con tarjeta.
  • Consulta el documento de reembolsos para más detalles específicos por país.

Prueba de la Integración

Prueba tu integración con credenciales de prueba antes de iniciar transacciones en producción.

  1. Inicia sesión en tu tienda, selecciona un producto para probar y procede al pago. Asegúrate de que el país de envío coincida con el destino:

  1. Selecciona el método de pago deseado:
  • Tarjeta: Ingresa los datos de la tarjeta y haz clic en Pay By Card.

Nota

Puedes personalizar el título de la opción de pago con tarjeta como se explica en Configuración de Métodos de Pago.
  • Pagar mediante Web Checkout: Para otros métodos de pago, selecciona Pay by Web Checkout y completa el pago.

  1. Una vez aprobado, verifica la compra en:
  • BigCommerce Order Dashboard: Ve a PayU Latam > Order Dashboard.

  • Panel de Gestión de PayU: Consulta el módulo de Reporte de Ventas.

Soporte

Para problemas técnicos o consultas sobre esta extensión, contacta a nuestro equipo de soporte en tecnico.co@payu.com o visita nuestro sitio web. Al contactar con soporte, incluye los detalles de la extensión en el asunto del correo y proporciona un resumen conciso del problema en el cuerpo del mensaje.
